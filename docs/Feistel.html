Class

Feistel

Feistel(key, rounds)

Feistel class is main point in Cipher Application.
This is implementation of Feistel Cipher
It supports all 65535 UTF-16 code points. Use encrypt() and decrypt() with appropriate text data.
You can read more about algorithm here: https://en.wikipedia.org/wiki/Feistel_cipher
